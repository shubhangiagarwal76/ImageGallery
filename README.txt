Image Gallery Windows Application

This application is developed on Visual Studio 2019 using C#. The purpose of this application is to search for required images and then export it to PDF. Images are fetched using Azure API. It will get maximum of 12 images per search. 

Features:
1. Image Search
2. Multi-Select Images
3. Export to PDF

Add On Features:
1. Auto-Complete/Suggestion Search Box
2. Gibberish search is handled
3. Selected Images Count on Status Bar
4. Topic of Search Displayed
5. Date Time at Status Bar
